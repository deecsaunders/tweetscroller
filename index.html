TweetScroller

About


TweetScroller is a simple jQuery plugin written in CoffeeScript that allows you to pull a collection of your favorite tweets using Twitter API v1 and display them in a fancy scroller with Masonry. Twitter API v1 is now deprecated. You'll need to use your own Twitter v1.1 proxy to use this plugin.

We created this for the Divshot lander and decided to open source it.

Dependencies


  • Handlebars
  • Moment.js
  • Masonry
  • hoverIntent

Markup


JavaScript


Option Default Value Description
url 'tweets.herokuapp.com?callback=?' URL to your Twitter API proxy. Must return a JSON response.
tweet '.tweet' Individual tweet <div> class.
template '#tweet-template' Handlebars template selector.
height 600 Height of container (in pixels).
speed 30 Speed of scroller in milliseconds. 30-80ms recommended.
autoplay false True: Autoscroll. False: Scroll on hover.

Proxy Example


This example uses Node.js and mtwitter to serve favorite tweets from the @divshot account. Tweets are cached for 1 hour. To generate your API key, visit Twitter Developers.
